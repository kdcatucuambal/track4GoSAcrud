# track4GoSAcrud
Example crud for track4GoSA using docker and docker-compose.
# Main Technologies Used
- Postgresql 12
- ReactJS
- NestJS
- Docker
# Commands and Ports
- docker-compose build .
- docker-compose up .
- localhost:3001 (back)
- localhost:3000 (front)
